# 00_Decision_Log

> Version: 1.0

詳細な決定事項およびその背景・理由は、個別のADRファイルを参照してください。

## ADRの記述方針

新しいアーキテクチャ上の決定を下す際は、以下のフォーマットに則って `ADR-XXXX_タイトル.md` を作成し、本ドキュメントに追記すること。

1. **Title** (タイトル)
2. **Status** (ステータス: 提案中 / 確定 / 却下 / 非推奨)
3. **Context** (背景・文脈)
4. **Decision** (決定事項)
5. **Consequences** (結果・影響)
